This forced me into building ModelForge ML, where I implemented classical ML algorithms from scratch—Linear Regression, K-Means, Decision Trees, SVMs—and benchmarked custom implementations against scikit-learn baselines.

📊 ModelForge ML Built a comprehensive ML algorithms library implementing gradient descent, eigendecomposition, and kernel methods from first principles. This project forced deep mathematical understanding instead of treating libraries as black boxes.

That foundation proved invaluable. I learned that mathematics matters, and more importantly, I realized my knowledge gaps couldn't be filled with MOOCs—they required hands-on, real-world constraints.

⚙️ Applied AI: From Agents to Enterprise Systems

RAG-Based Intelligent Agents

I built Clio, a retrieval-augmented generation (RAG) agent for historical analysis using LangChain, FAISS, Flask, and Google's Gemini API. This taught me where RAG pipelines excel—and critically, where they fail. I learned that retrieval quality gates the entire system's performance.

Full-Stack Production Systems

Then came RoleDrop Radar—a live, production-grade job monitoring platform that changed how I think about ML in engineering. This isn't a research project; it's a system serving 15–20 active users in real time.

🎯 RoleDrop Radar — Live Job Matching Platform
The Challenge: Semantically analyze thousands of job listings in real time while minimizing API costs.
The Solution: Hybrid regex + semantic pipeline that cut token consumption by 88% while maintaining classification accuracy—a real engineering trade-off, not a textbook problem.
Architecture:
  • Backend: FastAPI with pluggable source adapters (Ashby, Greenhouse, Lever, Google Careers)
  • Frontend: React + Vite with real-time telemetry dashboard
  • Database: Supabase (PostgreSQL) with Row-Level Security for multi-tenant isolation
  • AI: Gemini 2.5 Flash Lite with smart fallback to local Ollama models
  • Resilience: Circuit breakers, rate-limit failovers, self-healing pipelines

🎯 The Gaps I'm Here to Fill

My learning has been project-driven, which means I've often reached for working solutions without fully understanding the theory underpinning them. I know where my knowledge breaks down:

📐 Formal Optimization Theory

I use gradient descent, but I haven't rigorously studied convex optimization or convergence guarantees. I need deeper foundations in optimization landscapes and their implications for model training.

🎲 Probabilistic ML & Bayesian Methods

My experience is largely deterministic. Bayesian inference, latent variable models, and probabilistic graphical models remain underexplored areas I want to master for uncertainty quantification in production systems.

🧠 Advanced Deep Learning Theory

I've used transformers and diffusion models, but I want to understand attention mechanisms at a deep level, grok why certain architectures work, and understand diffusion model theory instead of just applying them.

🤖 Reinforcement Learning & Multi-Agent Systems

This is my biggest gap. I've built deterministic agent pipelines, but I haven't worked with policy optimization, multi-agent coordination, or learning from interaction. This is foundational for the agentic AI systems I want to build next.
